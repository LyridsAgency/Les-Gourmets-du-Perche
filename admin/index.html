Espace d'administration

Gérez le contenu de votre site : annonce, horaires, coordonnées, réseaux sociaux, réalisations et avis.

Comment obtenir ma clé d'accès ?
  1. Connectez-vous à GitHub puis ouvrez Settings → Developer settings → Fine-grained personal access tokens → Generate new token.
  2. Dans Repository access, choisissez Only select repositories et sélectionnez le dépôt du site.
  3. Dans Permissions → Repository permissions, passez Contents à Read and write.
  4. Générez le jeton et collez-le ci-dessus. Il reste enregistré uniquement dans ce navigateur.
